# smartui-cli

### Configure your Project Token

Setup your project token show in the **SmartUI** app after, creating your project.

 <b>For Linux/macOS:</b>
 
  ```
 export PROJECT_TOKEN="123456#1234abcd-****-****-****-************"
  ```

   <b>For Windows:</b>

  ```
  set PROJECT_TOKEN="123456#1234abcd-****-****-****-************"
  ```
